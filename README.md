# Hospital Management System (Frontend)

This is the **frontend** of a Hospital Management System built using **HTML, CSS, and JavaScript**.  
It provides a simple interface for patients, doctors, and appointments.

## Features
- Patient registration and management
- Doctor information and dashboard
- Appointment booking
- Responsive and user-friendly design
